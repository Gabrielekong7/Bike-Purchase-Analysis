# Bike-Purchase-Analysis
Project Overview

This project analyzes bike purchase behavior across different demographics, including age group, gender, and marital status. The objective is to identify key customer segments and uncover insights that can support better marketing and product decisions.

🔹 TOOLS USED
Microsoft Excel
Power Query
Pivot Tables
Power BI
Data Cleaning Techniques
🔹 Data Preparation
Cleaned the dataset by handling missing values and removing inconsistencies
Standardized categorical variables such as age groups and marital status
Transformed data using Power Query for better analysis
🔹 Analysis Performed
Examined bike purchases across different age groups
Analyzed the relationship between marital status and purchase behavior
Compared purchase patterns by gender
Visualized trends using an interactive Power BI dashboard
🔹 Key Insights
Middle-aged individuals represent the highest number of bike purchases
Married customers within the middle-age group are the most active buyers
Younger (adolescent) customers have lower current purchases but show potential for future growth
Male customers account for a higher proportion of bike purchases compared to females
🔹  WHAT I RECOMMEND,
The company should focus on  marketing strategies to middle-aged and married customers, as they are the primary buyers.
Develop targeted campaigns to attract younger customers and increase early adoption
Explore strategies to better engage female customers and improve their participation
Use customer segmentation to tailor product offerings and promotions
